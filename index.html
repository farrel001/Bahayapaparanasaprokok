⚠️⚠️⚠️ TUTORIAL: CARA NAMBAH RIWAYAT TRANSAKSI (UNTUK DEVELOPER) ⚠️⚠️⚠️
1
CARI ARRAY "transactions" DI DALAM SCRIPT

Gulir ke bawah sampai menemukan bagian yang bertuliskan "// ===== DATA TRANSAKSI ====="

2
FORMAT PENULISAN TRANSAKSI BARU

Setiap transaksi ditulis dengan format seperti ini:

// ⚠️ CONTOH TRANSAKSI (COPY PASTE INI UNTUK NAMBAH)
{
  date: "2025-03-20",    // ⚠️ Format: TAHUN-BULAN-TANGGAL
  desc: "Belanja di Pasar",  // ⚠️ Deskripsi transaksi
  category: "debit",      // ⚠️ Pilih: "credit" (masuk), "debit" (keluar), "transfer"
  amount: 500000,        // ⚠️ Jumlah dalam Rupiah (tanpa titik)
  status: "success",      // ⚠️ "success" atau "pending"
  icon: "fa-cart-shopping",  // ⚠️ Ikon Font Awesome
  warning: true          // ⚠️ KASIH ⚠️ di transaksi? true/false
},
3
LOKASI YANG TEPAT UNTUK NAMBAH

Tambahkan transaksi baru di dalam array, sebelum kurung siku tutup ]; Jangan lupa kasih koma setelah transaksi sebelumnya.

⚠️ PERHATIAN: Jangan hapus tanda kurung, koma, atau tanda petik yang udah ada, nanti error!
